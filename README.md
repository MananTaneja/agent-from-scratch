## Setup Instructions

This repo requires **Node.js version 20+**

The `main` branch contains the final application. To code along with the workshop, checkout the `step/1` branch. You will also need an API key

To install dependencies:

```bash
pnpm install
```

To run the project:

```bash
pnpm start
```
